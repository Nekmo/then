Audio
#####

:Component Type: Other
:Requirements: ffplay

Play a audio on local machine.


Setup for users
===============

These instructions are for the users. For developer instructions, look below.


Prerequisites
-------------

* A PC with support to play sound (speakers & sound card)
* Audio file to play
* ffplay installed


Config setup
------------

There are no config setup.


Message setup
-------------

These are the **required** parameters:

* **file**: Audio file path to play.
